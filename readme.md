# hotkeyz

A auto keyboard and mouse tool for windows os.

## Usage

### Use with Deno

```
deno run --allow-ffi --unstable .\deno_use.ts
```

### Use dynamic link library directly

build dll your self, and call exported functions as your wish.

## How to build

```
cargo build
```

## How to build doc

```
cargo doc --no-deps --open
```
